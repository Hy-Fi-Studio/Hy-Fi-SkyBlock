# Hy-Fi-SkyBlock
Hy-Fi SkyBlock provides a fully-featured SkyBlock experience tailored for Hytale. Players start on floating islands and expand through economy, farming, progression, and island upgrades. Supports team islands, coops, expansion unlocks, and resource automation. Works for grindy long-term servers.
